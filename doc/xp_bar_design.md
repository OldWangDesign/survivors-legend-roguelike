# 经验条 UI 设计文档

## 概述

经验条是游戏 HUD 中最显眼的元素之一，位于屏幕顶部全宽显示。好的经验条设计不仅传达进度信息，还应具有视觉吸引力和游戏反馈功能。

---

## 设计方案汇总

### 方案 A：斜切像素风

```
 ╱══════════════════════════════════════════╗
╱  EXP ████████████████░░░░░░░░░░░  Lv.3   ║
╚═══════════════════════════════════════════╝
```

- 左侧斜切 45° 角，右侧直角，形成"进攻感"箭头造型
- 填充条也是斜切的，像素锯齿边缘增加复古感
- **优点**：有冲击力，适合动作游戏
- **缺点**：斜切在低分辨率像素下可能锯齿明显

### 方案 B：分段宝石式

```
┌─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┐
│█│█│█│█│█│█│█│█│█│█│█│█│░│░│░│░│░│░│░│░│ Lv.3
└─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┘
      EXP 21/40
```

- 经验条分成 20 格（或根据升级所需动态分段）
- 每格填满时有个小闪光效果
- **优点**：非常复古（FC/SFC 风格），离散化显示让进度感更强
- **缺点**：分段数固定时精度有限

### 方案 C：双层装饰框

```
╔══╦══════════════════════════════════╦══╗
║☆║  ████████████████░░░░░░░░░  21/40 ║☆║  Lv.3
╚══╩══════════════════════════════════╩══╝
```

- 两端有像素装饰符号（星星/钻石/骷髅等）
- 粗像素边框，内部有高光和阴影层次
- **优点**：经典 RPG 对话框风格，精致感
- **缺点**：两端装饰占空间，信息密度稍低

### 方案 D：阶梯渐变条

```
▁▂▃▄▅▆▇█████████████░░░░░░░░░░  Lv.3
EXP 21/40
```

- 左边矮右边高，填充部分形成"阶梯上升"效果
- 高度从左侧 4px 渐变到右侧 16px
- **优点**：视觉独特，暗示"成长"，像素锯齿天然适配
- **缺点**：左侧较矮的部分文字不好放

### 方案 E：经典 JRPG 圆角槽

```
  ┌──────────────────────────────────────┐
◆ │▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓░░░░░░░░░░ 21/40│ Lv.3
  └──────────────────────────────────────┘
```

- 左侧菱形/宝石图标代表经验
- 条内填充有渐变色
- **优点**：像最终幻想/勇者斗恶龙的菜单，辨识度高
- **缺点**：相对传统，缺少独特性

### 方案 F：头像环绕弧形条

```
    ╭━━━━━━━╮
   ╱ ███░░░░ ╲
  │  [头像]   │  角色名  ★被动
   ╲ ░░░░░░░ ╱   Lv.3  21/40
    ╰━━━━━━━╯
```

- 经验条包裹在角色头像周围的弧形
- 从底部顺时针填充，满一圈升级
- **优点**：头像+经验合一，节省空间，视觉独特
- **缺点**：弧形在像素下绘制复杂，小尺寸可能模糊

### 方案 G：像素火焰条

```
         🔥
    ╱▓▓▓▓▓▓▓▓▓▓░░░░░░░░░░░░░░░╲
   ╱  EXP ▓▓▓▓▓▓▓▓▓░░░░░  Lv.3  ╲
```

- 填充条末端有 2-3 帧动画的像素火焰效果
- 颜色随填充百分比渐变：蓝 → 紫 → 金
- **优点**：动态感强，契合战斗主题
- **缺点**：火焰动画需要额外帧计算

### 方案 H：像素宝石槽

```
◇──◆──◆──◆──◆──◆──◇──◇──◇──◇──◇  Lv.3
         7 / 10 颗宝石
```

- 一排像素小宝石，获得经验逐颗点亮
- 点亮瞬间有小闪光动画
- **优点**：最复古的 RPG 感觉，视觉优雅
- **缺点**：宝石数量有限时精度低

### 方案 I：双轨道叠加条

```
┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓░░░░░░░░░░░░░░░░░┃ Lv.3
┃▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒                  ┃ 21/40
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

- 上轨道是当前经验（亮色），下轨道是残影（暗色）
- 获得经验时上轨道跳到新位置，下轨道缓慢追上
- **优点**：有"流动感"和获得经验的即时反馈
- **缺点**：双轨道占用更多垂直空间

---

## 最终采用：方案 J（组合方案）

组合方案 D（阶梯渐变）+ 方案 G（颜色渐变+火焰）的最佳元素：

### 视觉设计

```
  ▁▂▃▄▅▆▇████████████████░░░░░░░░░░░░  Lv.3
         ▓▓▓▓▓▓▓▓▓▓▓▓▓▓🔥              21/40
```

### 设计要素

| 特性 | 说明 |
|------|------|
| 造型 | 左低右高阶梯渐变，高度从 4px 到 18px |
| 颜色渐变 | 随填充进度：深蓝(0%) → 亮蓝(50%) → 紫色(80%) → 金色(95%+) |
| 火焰末端 | 填充条末端有 2 帧像素火焰/闪光动画 |
| 经验获取反馈 | 获得经验时填充区域短暂白色闪光脉冲 |
| 接近升级 | 进度 >90% 时整条微弱金色脉动 |
| 升级动画 | 整条闪金白光 + 高度短暂扩张弹跳 |
| 文字 | 右上 "Lv.3"（金色），右下 "21/40"（白色小号） |

### 颜色渐变细节

| 进度范围 | 主色调 | 说明 |
|----------|--------|------|
| 0% - 50% | 深蓝 → 亮蓝 | 初始阶段，稳定成长 |
| 50% - 80% | 亮蓝 → 紫色 | 中段加速，能量积累 |
| 80% - 95% | 紫色 → 金色 | 接近突破，视觉提醒 |
| 95% - 100% | 金色脉动 | 即将升级，闪烁提醒玩家 |

### 阶梯高度映射

经验条总宽度为屏幕宽度 - 16px（两侧 margin 8px）。高度按列从左到右线性增长：

- 最左列高度：4px
- 最右列高度：18px
- 每列宽度：1px（像素级绘制）

### 动画帧

**火焰末端（2帧交替，每帧 0.15s）：**

```
帧1:       帧2:
 ·          ·
·█·        · ·
█║█        ·█·
║║║        █║█
```

用 1-2 个像素在填充末端上方交替绘制，模拟火焰跳动。

### 选择此方案的理由

1. **视觉独特** — 阶梯造型在同类游戏中罕见，辨识度极高
2. **语义契合** — 从低到高暗示角色成长，经验积累的可视化隐喻
3. **像素风原生** — 阶梯锯齿是像素画的自然表达，不需要反锯齿
4. **动态丰富** — 颜色渐变 + 火焰 + 脉动 + 升级弹跳，层次感强
5. **实现友好** — 纯 `_draw()` 逐列绘制，无需额外贴图资源
